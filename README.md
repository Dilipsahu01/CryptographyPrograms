# CryptographyPrograms

A comprehensive repository demonstrating From-Scratch implementations of Cryptographic Algorithms and Hardware Logic.

## Contents
- **Python**: AES-128, DES, RSA
- **C**: AES-128, DES, RSA
- **Verilog**: Structural & Behavioral Full Adders, Hardware AXI4-Stream S-Box, Zero-Latency P-Box
- **Testing**: Automated test suite mapping to NIST test vectors.
